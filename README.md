<h2><b>Usage</b></h2>

```PowerShell
Install-Module cliHelper.core
```

then

```PowerShell
Import-Module cliHelper.core

$art = Create-CliArt "https://pastebin.com/raw/p29UR385" -Taglines "Build. Ship. Repeat."; $art.Replace("x.y.z", "0.2.4");
$art.Write(15, $false, $true)
```
